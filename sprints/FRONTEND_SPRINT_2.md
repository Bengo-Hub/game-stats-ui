# Frontend Sprint 2: Core Features  

**Duration**: 2-3 weeks
**Focus**: Dashboard, games, teams, players, live scoring

---

## Sprint Goals

- ✅ Dashboard with key metrics
- ✅ Game schedule and list views
- ✅ Team and player management
- ✅ Live scoring interface
- ✅ Search and filtering

---

## Tasks

### Week 1: Dashboard & Games

#### Day 1-3: Dashboard
- [ ] Create dashboard page (`app/(dashboard)/page.tsx`)
- [ ] Build metric cards:
  - Active games
  - Today's schedule
  - Top scorers
  - Recent activity
- [ ] Add quick actions
- [ ] Implement data fetching with React Server Components
- [ ] Add loading states

**Deliverable**: Interactive dashboard

---

#### Day 4-7: Games Views
- [ ] Build game list page:
  - Table view with filters
  - Card view option
  - Sort by date/status/division
- [ ] Create game detail page:
  - Team lineups
  - Current score
  - Game info (field, time, scorekeeper)
  - Scoring interface
- [ ] Build game schedule calendar view
- [ ] Add pagination
- [ ] Implement search

**Deliverable**: Complete game views

---
